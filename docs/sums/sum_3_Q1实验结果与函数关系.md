## 4. 函数关系式 (Ridge 全量拟合, R2=0.70)

```
log(1 + NTU) = 1.407
 - 0.287 * CLR
 - 0.262 * eta_coag
 + 0.255 * (-ln eta_coag)    ← 一级反应动力学项
 + 0.149 * FILT_NTU
 + 0.129 * FILT_NTU_mean3
 - 0.058 * CW_WELL_LEVEL_std12
 - 0.049 * CW_WELL_LEVEL
 - 0.047 * FILT_NTU_std12
```

物理含义:
- **CLR(-):** 色度高时NTU也高(正相关)
- **eta_coag(-) + (-ln eta)(+):** 两项联合描述去除效率的非线性效应, 效率越接近1→-ln(η)趋近0→出水越干净
- **FILT_NTU(+):** 滤后浊度直接决定出厂基线
- **FILT_NTU_mean3(+):** 近期趋势的正向叠加
- **CW_WELL_LEVEL相关项(-):** 清水池水位高→停留时间长→缓冲好→出水更干净

GAM偏依赖曲线(q1_gam_partial_dependence.png)验证各特征的加性边际效应光滑且单调, 与物理直觉一致。

## 6. 核心发现汇总

1. **FILT_NTU独占23% Shapley值** — 过滤单元是水质控制最后物理关口
2. **季节效应合计>原水水位** — 雨季/旱季是出厂水质的系统级驱动因素
3. **R2递进链: -0.54→0.07→0.34** — 非线性+交互本质确立
4. **NTU双峰分布: 87.6%稳态(mean=0.36) vs 12.4%冲击(mean=1.08)**
5. **RW_NTU→NTU直接r=0.04** — 99%处理效率致输入输出解耦
6. **物理幂次项- ln(η)进入TOP17** — 一级反应动力学假设获统计验证
7. **预定义L5交互全不显著** — 树模型内部已学, 预定义冗余
