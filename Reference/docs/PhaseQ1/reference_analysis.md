# Phase Q1 — 方法偏差对照与适配度评估

> 2026-07-24

## 1. 选型对照

| 选型决策 | 标准做法 | 本项目选择 | 偏差 | 判定 |
|------|------|------|------|:---:|
| 目标变换 | Box-Cox(λ=MLE) | log1p(NTU) | λ=-0.45改为λ=0 | 可接受(数值稳定性) |
| 特征筛选 | SHAP only | SHAP + Permutation ×10 + 几何平均 | 更严格 | 可接受 |
| 基模型 | 单XGBoost | XGBoost + LGBM + RF 三模型 | 增加了LGBM/RF | 可接受但集成无增益(LGBM调参后仍不敌XGB/RF) |
| 数据划分 | 随机5折 | TimeSeriesSplit(时序) | 更严格 | 正确 |
| 缺失列插补 | 均值/中位数 | LightGBM模型插补 | 偏差受控(R2≥0.71) | 可接受 |
| 特征工程 | 原始特征+SHAP筛选 | L1-L5五级金字塔+L2+幂次项 | 工程量大 | 正收益(L2:+0.44R2, 但L2+对XGB无用) |
| 函数关系 | 黑箱XGBoost | Ridge显式公式 + GAM偏依赖 | 增加可解释性 | 正收益(Ridge R2=0.70) |
| 消融 | 单层消融 | 8组逐层+重度消融(移除全部FILT17个特征) | 更完整 | 正收益(发现FILT占R2的42%) |
| LGBM调参 | 默认参数 | lr=0.01+L1L2+采样+400树 | R2: -0.01→0.27 | 正收益(但仍不及XGB/RF) |
| XGB/RF调参 | 默认参数 | 5组调参尝试 | 默认已最优 | 无需调整 |

## 2. 方法局限性

| 问题 | 影响 | 缓解 |
|------|------|------|
| MSE偏差 | NTU>0.5时系统性低估40%+ | Huber Loss(δ=1.0)部分缓解 |
| log1p非最优 | skew=3.63未完全正态化 | XGBoost对分布鲁棒 |
| 清水池缓冲效应不可见 | FILT高NTU低时严重误判 | Q2/Q3动机 |
| 2026年数据稀疏(12行/月) | L3/L4特征无法精确构造 | 2026仅做最终预测 |
| Format B 4个月缺失列 | ALUM/CL2插补引入噪声 | R2>0.7, 论文中声明 |
| L5交互特征不显著 | 4/4未通过检验 | 论文中报告为冗余 |

## 3. 与顶刊方法的差异

| 顶刊方法 | 本项目偏差 | 理由 |
|------|------|------|
| SHAP + 交叉验证 | 额外加了Permutation双重验证 | 保守策略, 鲁棒性优先 |
| 单模型最优 | 保留了三模型集成 | 论文叙事需要(渐进式) |
| 纯数据驱动 | 混合了物理幂次项(L2+) | 提升可解释性, 虽XGBoost无增益但Ridge有 |
| OOB验证 | 用的TimeSeriesSplit | 时序严格性 |
