## 5. T2: 双路径对比

| 路径 | FILT RMSE | R² | 说明 |
|---|---|---|---|
| A: 经验分布(基线) | 0.0363 | -0.9156 | 简单采样 |
| **B: 对数压缩灰箱** | **0.0289** | **-0.2410** | **k=0.10, α=4.99** |
| B: Sigmoid变体 | 0.0287 | -0.2202 | k=5.0, δ=0.10 |

**路径B公式** (最优: 对数压缩):
```
FILT_raw(t) = β₁·FILT(t-1) + (1-β₁)·RW_NTU(t)·[1 - η_base]
FILT_pred_T2 = 0.05 + k · log(1 + α · max(0, FILT_raw - 0.05))
```
其中k和α在T2验证集上优化获得。

R²为负的原因是T2区间FILT方差极小(0.026), 任何模型的误差规模都接近信号本身。**RMSE比经验分布低20%**, 说明对数压缩路径有实质改进。

NTU预测: 因FILT→NTU r=0.01(完全解耦), 使用条件经验分布 `NTU | FILT∈T2`。

## 6. T3: CSTR灰箱 + 负反馈调节

### 6.1 模型结构

```
FILT:  AR(6)  FILT(t) = 0.58·FILT(t-1) + 0.19·FILT(t-2) + ...

NTU:   CSTR段2  NTU(t) = β₂·NTU(t-1) + (1-β₂)·FILT(t) + γ(t)·ε_alum(t)
         β₂ = exp(-2h / θ),  θ = A · CW_WELL(t-1) / TW_FLOW(t-1)

反馈:  ε_alum(t) = ALUM(t) - ALUM_policy(t)
         ALUM_policy(t) = Linear(RW_NTU, log(RW_NTU), FILT_lag1, month)

γ(t):  sigmoid(γ_w·RW_NTU(t) + γ_b) - 0.5    (时变反馈增益)

Loss:  Huber + 0.1·平滑 + 0.5·上界(ReLU(NTU-RW)) + λ₃·|ε_alum|
```

### 6.2 T3扫参结果

Top 5配置:

| fb | γ | λ₃ | NTU R² | RMSE | τ₁(peak) |
|---|---:|---:|---:|---:|---:|
| linear | sigmoid | 0.50 | **0.6475** | 0.616 | 4h |
| linear | sigmoid | 0.05 | 0.6345 | 0.635 | 6h |
| none | linear_rw | 0.05 | 0.6309 | 0.628 | 8h |
| linear | linear_rw | 0.05 | 0.6081 | 0.655 | 8h |
| linear | sigmoid | 0.00 | 0.5954 | 0.657 | 2h |

### 6.3 全量NTU验证 (CSTR段2)

```
NTU(t) = β₂·NTU(t-1) + (1-β₂)·FILT(t),  β₂=exp(-2h/θ),  θ=A·CW_WELL/TW_FLOW
```

| 方法 | 全量 R² | 全量 RMSE | CV R² | T1 R² | T2 R² | T3 R² |
|---|---:|---:|---:|---:|---:|---:|
| Pure CSTR (A=141.3) | **0.7272** | 0.3445 | 0.7318 | 0.862 | 0.757 | 0.668 |
| CSTR+Feedback | 0.7287 | 0.3436 | — | 0.858 | 0.756 | 0.671 |
| XGBoost(原101维) | 0.3400 | 0.437 | — | — | — | — |

**CSTR物理模型在全量数据上R²=0.727, 远超原XGBoost黑箱的0.34, 提升+0.39。**

### 6.4 τ₁可学习 (softmax加权)

τ₁权重分布: [0.006, 0.035, **0.362**, **0.361**, 0.063, 0.092, 0.081]

物理含义: RW_NTU→FILT的最优时滞为**4h(2步)**, 与物理先验(混凝30min+沉淀2h+过滤30min=约3-4h)一致。

### 8.3 函数关系 (CSTR物理公式)

**主推 — CSTR递推公式 (R²=0.727)**:
```
NTU(t) = exp(-2h/θ) · NTU(t-1) + [1 - exp(-2h/θ)] · FILT(t)
其中 θ(t) = A · CW_WELL_LEVEL(t-1) / TW_FLOW(t-1),  A=141.3
```

**辅助 — Ridge线性近似 (R²=0.70)**:
```
log(1+NTU) = 1.407 - 0.287·CLR - 0.262·η_coag + 0.255·(-ln η) + 0.149·FILT_NTU
            + 0.129·FILT_NTU_mean3 - 0.058·CW_WELL_LEVEL_std12 - 0.049·CW_WELL_LEVEL - 0.047·FILT_NTU_std12
```

## 10. 核心创新点

1. **三级FILT分区**: 将数据按FILT_NTU自然分为噪声区(49%)/过渡区(30%)/应力区(21%), 每个区域采用不同策略
2. **CSTR适用于NTU而非FILT的发现**: 清水池混合效应由NTU递推描述, FILT自身是AR过程
3. **τ₁可学习(softmax)**: 跳过传统统计时滞估计(CCF/MIC/TE全部失效), 端到端学习得到τ₁=4h
4. **操作员反馈建模**: 线性策略模型分解控制变量, sigmoid γ(t)实现时变反馈增益
5. **η_coag为应力区核心因素**: 滤池穿透时混凝去除效率取代滤后值成为首要预测因子
6. **经验分布优于高斯**: T1中FILT数据为离散值, 经验采样JS=0.05远优于高斯0.64
