## 5. 不可变决策

1. **Q1不再使用XGBoost/SHAP/三模型集成** — 已废弃为三级灰箱CSTR
2. **Q2不再使用全量TCN/Attention/PINN** — 已废弃为双模诊断
3. **特征矩阵仅~12维** — 不再恢复L2+/L3/L4/L5层
4. **泵编码为2维count** — 不再恢复10维0/1展开
5. **CSTR适用于NTU, 不适用于FILT** — FILT(t)是AR过程, NTU(t)是CSTR递推
6. **物理约束只在违规数据上激活** — 违规率0%时用硬裁剪
7. **统计时滞估计不再使用** — CCF/MIC/TE全部失效, 改用softmax可学习τ₁
8. **Reference/ 位于 git 根目录内 (`Code/Reference/`)** — 随代码一同版本控制
5. **全部模型验证在2025数据TS-CV** — 2026仅做最终预测
6. **NTU空间工作(非Box-Cox空间)** — 灰箱模型不依赖变换

## 5. 测试纪律

- 每新建step文件 → 运行完整脚本验证
- 每修改step文件 → 重新验证影响链
- 变动任一config.py常量 → 全量重跑受影响文件
- 每次提交前 → 确保所有输出文件可正常读取
